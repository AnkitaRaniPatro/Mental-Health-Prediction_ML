# 🧠 Mental Health Prediction in Tech Industry
## 📌 Project Overview
This project predicts whether a person will seek mental health treatment using machine learning based on workplace and personal factors.

## 🎯 Objective
Build a model to identify employees who may need mental health support.

## 📊 Dataset
* OSMI Survey
* 1259 rows, 27 features
* Target: Treatment (Yes/No)

## ⚙️ Technologies Used
* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* XGBoost

## 🔍 Workflow
* Data Cleaning
* EDA
* Feature Engineering
* Model Training
* Evaluation

## 📈 Results
* Accuracy: 78%
* ROC-AUC: 85%

## 💡 Insights
* Family history is important
* Work interference affects decisions
* Company support matters

## ▶️ How to Run
1. Install libraries:
   pip install pandas numpy scikit-learn matplotlib seaborn xgboost
2. Run:
   jupyter notebook
